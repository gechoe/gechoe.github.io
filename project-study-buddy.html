StudyBuddy

An Android & Web Application to boost motivation during study sessions.

Project Overview

StudyBuddy is an application designed to help students stay motivated during their study sessions by introducing collectible virtual companions called StudyBuddies.

Key Features:

  • Users can earn StudyBuddy companions by completing timed study sessions.
  • Available on both Android and Web platforms.
  • Built using Java, Android Studio, JavaScript, and HTML.
View on GitHub

Technologies Used

Java, Android Studio, JavaScript, HTML, Firebase

Real-Time Sync

Study sessions are tracked and synced in real-time with the database using Firebase.

Customizable UI

Users can choose from different themes to customize their StudyBuddy's appearance.

Responsive Design

The web version is fully responsive and optimized for both desktop and mobile users.
